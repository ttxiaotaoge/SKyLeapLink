# Skills - LLM元技能系统

将你的日常工作转化为AI智能代理，通过系统化的技能定义，让LLM帮助你自动化工作流程。

## 📋 目录结构

```
skills/
├── README.md                 # 本文件 
├── meta-skills.md           # 元技能指导框架
├── categories.md            # 技能分类体系
├── templates/               # 技能定义模板
│   ├── basic-skill.md      # 基础技能模板
│   └── workflow-skill.md    # 工作流技能模板
├── examples/                # 实际技能示例
│   └── [你的技能].md
└── workflows/               # 工作流程编排
    └── [自动化流程].md
```

## 🎯 核心概念

### 什么是Skills？
Skills 是你与AI交互的**接口层**。它把你的日常工作、知识技能、业务流程转化为结构化的指令，让LLM能够理解和执行。

### 三层架构

```
┌─────────────────────────────────────┐
│        你的日常工作和知识             │  (Daily Work & Knowledge)
├─────────────────────────────────────┤
│     通过Skills定义梳理和表达          │  (Skills Definition Layer)
├─────────────────────────────────────┤
│    LLM智能代理执行和自动化             │  (LLM Agent Execution)
└─────────────────────────────────────┘
```

## 🚀 快速开始

### 第一步：梳理你的技能

1. 打开 [meta-skills.md](meta-skills.md) 了解如何系统地定义技能
2. 根据 [categories.md](categories.md) 对你的工作进行分类

### 第二步：使用模板定义技能

选择合适的模板：
- **基础技能模板** - 用于单一、独立的工作任务
- **工作流技能模板** - 用于多步骤、跨职能的复杂流程

### 第三步：创建你的技能文件

在 `examples/` 文件夹中创建你的技能定义文件。

### 第四步：与LLM交互

将你的skills提交给LLM，让它：
- 理解你的工作方式
- 学习你的决策规则
- 自动执行和优化流程

## 📚 技能类别

| 类别 | 说明 | 示例 |
|-----|-----|------|
| **工作技能** | 日常完成的具体任务 | 编写报告、数据分析、客户沟通 |
| **知识技能** | 专业知识和能力 | 法律咨询、代码审查、架构设计 |
| **流程技能** | 多步骤综合流程 | 项目启动、hiring流程、产品发布 |
| **决策技能** | 基于规则的决策 | 优先级判断、需求评审、绩效评估 |

## 🛠️ 工作流程

```
1. 识别 (Identify)
   ↓
2. 分解 (Decompose) 
   ↓
3. 定义 (Define)
   ↓
4. 验证 (Validate)
   ↓
5. 部署 (Deploy)
   ↓
6. 优化 (Optimize)
```
