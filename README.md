把世间万物连接你的LLM大脑终端，实现数字永生

## 🎯 项目内容

### [📚 Skills - LLM元技能系统](skills/)

这是本项目的核心模块，帮助你系统地：
1. **梳理日常工作** - 识别哪些工作可以自动化
2. **定义关键技能** - 将工作转化为AI可理解的"技能"
3. **交给AI执行** - 让LLM智能代理帮你完成重复性工作
4. **持续优化** - 基于经验迭代改进规则

#### 🚀 快速开始
```bash
# 在3步内开始
1. 打开 skills/README.md
2. 选择一个常做的工作
3. 按照 skills/QUICK_START.md 花30分钟定义它
4. 交给LLM，让它帮你做！
```

#### 💡 使用场景

```
适合自动化的工作：
✅ 日报、周报汇总

✅ 会议记录整理

不适合的工作：

折中方案：

⚠️ AI提供建议，专家做决策
```
